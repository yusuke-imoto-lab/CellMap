# CellMap - RNA landscape inference method

The CellMap is a RNA landscape inference method based on the graph Hodge decomposition. 

## Installation

To install CellMap package, use `pip` as follows:

```
$ pip install cellmap
```

## Requirements
* Python3
* numpy
* scipy
* scikit-learn
